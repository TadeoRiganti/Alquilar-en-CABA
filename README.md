### Funcionalidades

- Mapa interactivo.
- Visualizacion de datos relacionados a CABA:
  - Urbanismo:
    - Comunas y barrios.
    - Villas y asentamientos informales.
  - Movilidad:
    - Estaciones de SUBTE.
    - Estaciones de Ferrocarril.
  - Estadisticas de Delito:
    - Robo total.
    - Robo automotor.
    - Hurto total.
    - Hurto automotor.
